# Air-Scribbling-using-OpenCV

This project enables real-time hand gesture drawing by leveraging computer vision tracking and rendering. It allows users to create drawings or scribbles in the air using their hand movements.

Project Overview
The core idea is to capture real-time hand movements via a webcam. OpenCV then processes these video frames to detect the position of the hand, converting these detected movements into a virtual drawing or scribble on the screen.

Features
Real-time Hand Tracking: Utilizes computer vision to continuously track hand movements.
Air Drawing: Allows users to "draw" in the air using their hands.
Virtual Rendering: Translates physical hand movements into digital output.
Technologies Used:
  • Python 
  • Machine Learning (ML) 
  • OpenCV
